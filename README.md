## 📋 О проекте

MebelFlow автоматизирует бизнес-процессы мебельного центра: от замера до сдачи отчетности.

## 🚀 Функциональность

- Управление заказами (создание, просмотр, статусы)
- Финансовый модуль (предоплата 50%, остаток долга)
- Отчетность (выручка, налог УСН 6%)
- Ролевая модель (дизайнер, менеджер, бухгалтер, администратор)
- Графический интерфейс (Tkinter)
